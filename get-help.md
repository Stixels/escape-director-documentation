# Get Help

Use this page to decide the fastest way to get the kind of help you need.

## Use Documentation When

Use the docs when you need:

* Setup instructions
* Product how-to guides
* Operational guidance for rooms, Live View, and Room Statistics
* Troubleshooting steps for common issues

Start with:

* [Getting Started](getting-started.md)
* [Build Your Rooms](build-your-rooms/README.md)
* [Run Live Sessions](run-rooms/README.md)
* [Review Performance](review-performance/README.md)

## Use The Feedback Portal When

Use the feedback portal when you want to:

* Submit feature requests
* Review the roadmap
* Read product updates and changelog items
* Track improvement ideas over time

## Use In-App Chat When

Use in-app chat when you need direct help from inside the product and want to explain a current issue quickly.

Chat is best for:

* Questions that are blocking current work
* Clarification on how a shipped feature is supposed to behave
* Help identifying the next troubleshooting step

## Support Email

If you need direct support by email, contact [support@escapedirector.com](mailto:support@escapedirector.com).

## Before Contacting Support

Include as much of this as possible:

* Room name
* What page or feature you were using
* Browser and device
* What you expected to happen
* What actually happened
* What steps have you taken to try to fix an issue
* Screenshot or screen recording if available
* Whether the issue affected Live View, media, cache status, or Room Statistics

The clearer the report, the faster the issue can usually be understood and reproduced.
