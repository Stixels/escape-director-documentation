# 👋 Welcome to Escape Director!

## What is Escape Director?

[Escape Director](https://www.escapedirector.com/) (ED) is a control center for escape room operations. It’s built to run rooms the best way. Fast onboarding. Clean execution. Better decisions.

If you’re tired of scattered notes and “tribal knowledge”. ED makes every run repeatable and every improvement measurable.

### Why teams choose ED

* **Faster onboarding:** new game masters learn one system, not ten docs.
* **Rich media clues:** deliver images, video, and audio from the same workflow.
* **Easier tracking:** every run has a full log of events and actions.
* **Data-driven ownership:** stats + logs show what to fix and what to keep.

### Built for

* **Owners** who want consistency across staff and locations.
* **Designers** who want structure without killing creativity.
* **Game masters** who need speed and clarity during live runs.

### What you can do with ED

* Build rooms with puzzles, hints, and rich media clues.
* Run sessions with a game master friendly workflow.
* Capture **full logs** for every run.
* Use statistics to improve rooms and coach staff.

{% hint style="info" %}
ED runs in your browser. No installs. No manual updates.
{% endhint %}
