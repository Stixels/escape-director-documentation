# ✍️ Editing Rooms

## Editing Rooms

Open **All Rooms**, then click **Edit Room** on a room card. Use the editor tabs to update room settings, puzzles, and media.

### Tab-by-tab details

{% tabs %}
{% tab title="Room Details" %}
Use this tab to configure the room defaults.

* Room name
* Default time
* Default clues / hints
* Any other room-level settings

Click **Save Room Details** when you’re done.
{% endtab %}

{% tab title="Puzzles & Clues" %}
Use this tab to create and manage puzzles, plus the hints/clues tied to them.

1. Click **Add Puzzle**.
2. Fill in the puzzle details.
3. Add hints/clues in the order you want to run them.
4. Click **Save** to persist your changes.

{% hint style="info" %}
Puzzles may auto-save while you edit, but use **Save** as your final step.
{% endhint %}
{% endtab %}

{% tab title="Images" %}
Upload and manage images used during runs.

1. Add an image in the upload area.
2. Click **Upload**.
3. Use 🗑️ to delete an uploaded image.
{% endtab %}

{% tab title="Videos" %}
Upload and manage videos used during runs.

1. Add a video in the upload area.
2. Click **Upload**.
3. Use 🗑️ to delete an uploaded video.
{% endtab %}

{% tab title="Audio" %}
Upload and manage audio used during runs.

1. Add an audio file in the upload area.
2. Click **Upload**.
3. Use 🗑️ to delete an uploaded audio file.
{% endtab %}
{% endtabs %}
