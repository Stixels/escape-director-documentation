# Add Puzzles And Clues

Puzzles and clues define how your room is run inside Escape Director. This is where you set the structure game masters will use during a live session.

## Before You Begin

Make sure you have:

* Created and configured the room
* Opened the room in the editor
* Finished the basic room setup

## What You Can Configure

| Field              | Purpose                                                 |
| ------------------ | ------------------------------------------------------- |
| Puzzle name        | Identifies the puzzle for your team                     |
| Answer or solution | Stores the expected answer or solve condition           |
| Description        | Adds internal context for staff                         |
| Clues              | Stores the clue text your team may send during the game |

You can also reorder:

* The puzzles in the room
* The clues inside each puzzle

## Open The Puzzles Editor

1. Go to `All Rooms`.
2. Open the room you want to edit.
3. Click `Edit`.
4. Open the `Puzzles` tab.

## Add A New Puzzle

1. Click `Add New Puzzle`.
2. Enter the puzzle details.
3. Save the puzzle.

### Puzzle Details

#### Puzzle name

Use a clear name your game masters will recognize immediately during a live run.

#### Answer or solution

Use this for the expected answer, code, password, key, or other solve condition.

#### Description

Add a short internal description when staff need extra context.

## Add Clues To A Puzzle

Each puzzle can include one or more clues. Build clues in escalation order so game masters can move from subtle guidance to direct help.

| Clue stage  | Example                                             |
| ----------- | --------------------------------------------------- |
| Early hint  | Take a closer look at the objects near the map.     |
| Mid hint    | One of the objects near the map can be opened.      |
| Direct hint | Use the small key on the wooden box beside the map. |

{% hint style="info" %}
Ideally, game masters should not need to write custom clues during a session. Good clue setup makes live operation faster and more consistent.
{% endhint %}

## Reorder Puzzles And Clues

Arrange puzzles in the order your team expects to follow during the game. Arrange clues from lighter guidance to more direct assistance.

This makes it easier to:

* Find the right puzzle quickly
* Deliver clues consistently
* Keep the room structure clean

## Review The Puzzle List

Before live use, make sure:

* Puzzle names are clear
* Answers are present where needed
* Descriptions are useful, not repetitive
* Clues are written clearly
* Puzzle order matches the intended room flow

Remove old or unused entries instead of leaving them behind.

## Best Practices

* Name puzzles for operations, not just internal design.
* Write clues in escalation order.
* Keep clue text concise.
* Remove duplicate or weak clues.

## Troubleshooting

### I am not sure how many puzzles to add

Start with the real operational puzzle sequence your team already uses. Refine it later if needed.

### Should every puzzle have clues?

Not necessarily. Add clues where staff may realistically need them during gameplay.

### Can I change puzzle order later?

Yes. Reorder puzzles any time if the room flow changes.

### Should I store full game master instructions in the clue field?

No. Use clue fields for text your team may actually send during a game.
