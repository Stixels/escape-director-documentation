# Create And Configure Your First Room

<figure><img src="../.gitbook/assets/Edit Room 1.png" alt=""><figcaption><p>Room Edit Page Core Configuration</p></figcaption></figure>

Creating a room is the first real setup step inside Escape Director. Once the room exists, you can configure its defaults, add content, and prepare it for live sessions.

## Before You Begin

Make sure you have:

* Signed in to Escape Director
* Activated your trial or plan
* A basic idea of the room you want to set up

You do not need every detail finalized before creating the room. You can edit the room later.

## Create A Room

1. Open the `All Rooms` page after signing in.
2. Click `Create Room`.
3. Enter the basic room information.
4. Save the room to continue into the editor.

### Basic Room Information

| Field | Purpose |
| --- | --- |
| Room name | Internal room name for staff |
| Time limit | Default start time for the room |
| Default clue allowance | Default number of hints available to players |

{% hint style="info" %}
You can add or remove time and clues later during a live game.
{% endhint %}

{% hint style="info" %}
If your team does not use a fixed clue limit, start with your normal default or `0` and adjust during the game.
{% endhint %}

## Continue Into The Editor

After the room is created, Escape Director opens the room editor so you can keep building it.

Creating the room gives you the starting shell. The rest of the setup happens in the editor.

## Best Practices

### Use a general puzzle

Start with a general puzzle for common messages such as hint offers or standard in-room reminders.

### Use names your staff will understand instantly

Operational clarity matters more than decorative naming inside the app.

### Test before going live

Run an internal test session so staff can confirm:

* Timer behavior
* Puzzle flow
* Clue usage
* Media playback
* Live View behavior

{% hint style="warning" %}
Test the room before live use so game masters can catch missing clues, broken media, or layout issues early.
{% endhint %}

## Troubleshooting

### I created the room, but it looks incomplete

That is expected. Room creation only gives you the base room. You still need to configure details, add puzzles, and upload media.

### I entered the wrong room details

Open the room in the editor and update the details there.

### I cannot find the room after creating it

Return to `All Rooms`, refresh the page, and check again. A hard reload can help if the list looks stale.

### Should I delete the room and start over if I made a mistake?

Usually no. In most cases, it is better to edit the existing room.

{% hint style="danger" %}
Deleting a room removes its associated media, clues, and statistics.
{% endhint %}
