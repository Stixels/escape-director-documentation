# 🚪 Managing Escape Rooms

<figure><img src="../../.gitbook/assets/App Home.png" alt=""><figcaption><p>Example of the All Rooms Page</p></figcaption></figure>

## Managing Escape Rooms

After you log in, you land on the **All Rooms** page. This is your hub for creating rooms and jumping into a live dashboard.

### What you’ll see on the All Rooms page

* Room cards for each escape room.
* A **Create Room** button.
* Actions on each room (run, edit, delete).

### Creating a room

1. Click **Create Room**.
2. Enter the basic room info.
3. You’ll be taken to **Edit Room** to add content.

Next: [Editing Rooms](editing-rooms.md).

### Editing a room

From **All Rooms**, click **Edit Room** on a room card.

In the editor, you can:

* Customize default messages.
* Add and reorder puzzles.
* Add hints per puzzle.
* Attach rich media (images, video, audio) for clue delivery.

Full walkthrough: [Editing Rooms](editing-rooms.md).

### Deleting a room

On the room card, click the 🗑️ icon. Confirm the prompt to delete the room.

{% hint style="danger" %}
Be careful deleting rooms. This will delete all data associated with the room as well.
{% endhint %}

### Running a room

Click a room card to open the **Room Dashboard**. This is where game masters run sessions and control the Live View.

Next: [Running Escape Rooms](../running-escape-rooms.md).
